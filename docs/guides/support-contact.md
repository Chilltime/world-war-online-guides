# Contact & Support

Need help with World War Online? This page provides all the ways to get support and contact the team.

## Getting Help

### In-Game Support

For issues within the game:

1. Access the support section from the game menu
2. Submit a ticket describing your issue
3. Include relevant details (account name, what happened, when)
4. Wait for a response from the support team

### Community Support

Connect with other players for help:

| Channel | Best For |
|:--------|:---------|
| **Global Chat** | Quick questions, finding squads |
| **Discord** | Detailed discussions, community help |
| **Squad Chat** | Team coordination, squad-specific issues |

## Common Issues

### Account Issues

| Issue | Solution |
|:------|:---------|
| **Login problems** | Reset password, check email verification |
| **Lost progress** | Contact support with account details |
| **Account recovery** | Submit support ticket with proof of ownership |

### Gameplay Issues

| Issue | Solution |
|:------|:---------|
| **Battle bugs** | Report with screenshots/details |
| **Missing rewards** | Check all collection points, then report |
| **Squad issues** | Contact squad leader first, then support |

### Purchase Issues

| Issue | Solution |
|:------|:---------|
| **Missing WWO** | Wait a few minutes, then contact support |
| **Payment failed** | Check payment method, try again |
| **Refund requests** | Review [Store Rules](../legal/store-rules.md), contact support |

## Reporting Players

To report rule violations:

1. Document the violation (screenshots help)
2. Submit a report through the in-game system
3. Include player name and specific violation
4. Allow time for investigation

See [Reports, Bans & Suspensions](../legal/reports-bans-suspensions.md) for details.

## Feedback & Suggestions

Have ideas to improve the game?

- Share in Discord suggestion channels
- Discuss with the community
- Well-articulated suggestions help development

## Before Contacting Support

To help resolve your issue faster:

1. **Check the guides** - Your answer may already be documented
2. **Search FAQ** - Common questions are answered there
3. **Be specific** - Include account name, time, and details
4. **Be patient** - Support handles many requests
