# Battalion Support Rules

This guide covers the specific rules and restrictions for Battalion Support in World War Online.

## Basic Requirements

To provide Battalion Support, you must meet these conditions:

| Requirement | Description |
|:------------|:------------|
| **Battalion membership** | Must be in the same battalion as the player you're supporting |
| **Proximity** | Must own bases within 4 districts of the target or defended base |
| **Same continent** | Support only works within the same continent |

## Unit Limits

### Support Budget

Support units are limited to **30% of the district's unit capacity**. This budget is shared among all supporters.

| District Type | Normal Units | Supreme Units |
|:--------------|:------------:|:-------------:|
| Regular District | 900 (30% of 3,000) | 22 (30% of 75) |
| Headquarters District | 1,200 (30% of 4,000) | 30 (30% of 100) |

### Unit Group Limits

- Maximum **2 unit groups** per support action
- Per-group limits: **750 Normal units**, **25 Supreme units**

## Restricted Units

These units cannot be used for Battalion Support:

| Unit Type | Allowed? |
|:----------|:--------:|
| Normal Units | Yes |
| Supreme Units | Yes |
| Generals | No |
| Special Forces | No |

Only Normal and Supreme combat units are allowed in support actions.

## Timing

- Support units join the battle at **wave 10**
- Defense Support units are **hidden** from attackers until arrival
- Support strategy bypasses normal deployment phases

## Attack Support Rules

| Rule | Description |
|:-----|:------------|
| **Distance** | Must own a base within 4 districts of the attack target |
| **Continent** | Must be on the same continent as the target |
| **Strategy** | Uses Attack Support strategy only |
| **Arrival** | Units join the attack at wave 10 |

## Defense Support Rules

| Rule | Description |
|:-----|:------------|
| **Distance** | Must own a base within 4 districts of the defended base |
| **Continent** | Must be on the same continent as the defended base |
| **Visibility** | Defense support units are hidden until wave 10 |
| **Strategy** | Uses Defend Support strategy only |
