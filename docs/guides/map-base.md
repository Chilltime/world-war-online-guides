# Base View

The Base View provides detailed information about a specific base and allows you to take actions like attacking, upgrading, or training units.

## Accessing Base View

Click on any base in the [District View](map-district.md) to open its detailed view.

## Your Own Bases

When viewing your own bases, you can:

### Available Actions

| Action | Description |
|:-------|:------------|
| **Upgrade** | Increase base level (costs Brick) |
| **Train Units** | Produce new units at this base |
| **Manage Defense** | Configure defensive positioning |
| **View Production** | See resource output rates |

### Base Information

- **Base Type** - Headquarters, Resource, Mine, Military, or Capital
- **Base Level** - Current level (1-10)
- **Production Rate** - Resources generated per hour
- **Upgrade Cost** - Brick required for next level
- **Upgrade Time** - Duration for next upgrade

## Enemy Bases

When viewing enemy bases, you can:

### Available Actions

| Action | Description |
|:-------|:------------|
| **Attack (Raid)** | Send units to steal resources |
| **Conquer** | Send General to take ownership |
| **Destroy** | Send Special Forces to remove base |

### Scouting Information

- **Base Type** - What kind of base it is
- **Owner** - Which player controls it
- **Potential Resources** - What you might steal
- **Attack Availability** - Whether you can attack (cooldown status)

## Base Type Tooltips

Hover over any base to see a tooltip showing:

| Base Type | Tooltip Shows |
|:----------|:--------------|
| **Headquarters** | HQ level, owner |
| **Resource Base** | Resource type (Food, Iron, Money, Brick), level |
| **Mine Base** | Iron production, level |
| **Military Base** | Special Forces training, level |
| **Capital Base** | Capital warfare status |

## Strategic Use of Base View

### Before Attacking

1. Check base type to understand what you can gain
2. Assess if conquest is possible (check your base limits)
3. Decide between raid, conquer, or destroy

### Base Management

1. Prioritize upgrades based on your strategy
2. Balance production across your bases
3. Configure defenses before threats arrive
